# Friend data parser

## Dependencies

```buildoutcfg
pip install riotwatcher
pip install pandas
pip install tqdm
```

## Run a script

```buildoutcfg
--api-key RGAPI-xxx 
--region eun1 --my-summ your_summoner_name 
--my-friends friend1 friend2 friend3 ...
--max-match-count 100
```

Navigate to [Riot games dev portal](https://developer.riotgames.com/) to get your API key.
